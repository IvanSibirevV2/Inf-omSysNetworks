Адрес ICD AESA

Формат адреса ICD AESA схож с форматом DCC, за исключением следующих моментов:

  • поле AFI равно 47, а не 39;
  • поле IDI содержит ICD (International Code Designator) — двухбайтовый идентификатор организации, который выдается Британской организацией по стандартизации, а не отделением ISO
